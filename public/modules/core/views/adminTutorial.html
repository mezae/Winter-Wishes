Ready, Go!

This page will help you easily create and manage tracking forms for each agency accepted into the Winter Wishes program this year. Let's get started!

First, let's set the due date for the tracking forms:

In order to proceed, all Winter Wishes applications in salesforce should be processed, and you should have the following information for each accepted agency:

  • Agency Code
  • Agency Name
  • Contact Name
  • Contact E-mail
  • Accepted Children
  • Accepted Teens
  • Accepted Seniors

Download a salesforce report with these fields, and save it as a csv file.

Drag the csv file into the dropzone or click the dropzone to use the file browser. If all necessary data is found, the accepted agencies will appear onscreen within seconds. Each agency will be represented like this:

ELM

100 Letters

Hover over it to see the three possible actions.

Click in the box of the tracking form you wish to edit. A form will appear populated with the current information for the agency you selected. Once you have made the necessary modifications, click to safeguard your changes.

* If you forgot to add an agency, just click for a blank form.

Click in the box of the tracking form you wish to delete. This will delete everything associated with the agency. This cannot be undone. For this reason, you will be asked to confirm that you really want to delete the selected agency.

Click in the box of the tracking form you wish to see. You can access an individual tracking form anytime to check how much progress a particular agency contact has made. However, this feature is more useful once an agency has submitted their form and you are ready to review it. You will know when tracking form is ready for review because the color of the box will change based on what stage of the process the tracking form is in.

Once all the tracking forms have been created, you will need to e-mail all accepted agencies to let them know that they have been accepted into this year's Winter Wishes program and give them further instructions. For help with that, head over to the E-mail Templates via the main menu.

You can also access this tutorial on the main menu if you need to see it again.
